Student Management Portal - README
Project Overview

Student mangement portal a project based on java learning mainly:

JDBC (Java Database Connectivity): Connecting your Java application to a relational database (like MySQL or PostgreSQL) to perform CRUD (Create, Read, Update, Delete) operations.

Multithreading and Concurrency: Writing code that can execute multiple tasks simultaneously (useful for performance, handling multiple users, or background tasks).

Lambda Expressions & Stream API: Modern Java features (since Java 8) for writing more concise and functional code, especially when working with collections.

Frameworks: Using libraries and frameworks to accelerate development, JavaFX/Swing (for desktop GUIs).

Tech Stack
- Java (JSP, Servlets)
- JDBC
- MySQL
- Apache Tomcat
- HTML, CSS, Bootstrap
Features
- Admin Login
- Student Login
- Add, Edit, Delete Students
- View Student List
- Dashboard Stats
- Student Dashboard
Database Tables
1. students(id, name, course, marks, attendance, password)
2. admin(username, password)
How to Run
1. Import project into Eclipse
2. Add MySQL Connector to WEB-INF/lib
3. Configure Tomcat Server
4. Create MySQL database and tables
5. Run the project and open http://localhost:8080/StudentPortalWeb
