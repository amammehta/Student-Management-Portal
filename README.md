Student Management Portal - README
Project Overview
This is a Java-based Student Management Portal built using JSP, Servlets, JDBC, and MySQL. It supports Admin and Student login, CRUD operations, dashboards, and authentication.
Tech Stack
- Java (JSP, Servlets)
- JDBC
- MySQL
- Apache Tomcat
- HTML, CSS, Bootstrap
Features
- Admin Login
- Student Login
- Add, Edit, Delete Students
- View Student List
- Dashboard Stats
- Student Dashboard
Database Tables
1. students(id, name, course, marks, attendance, password)
2. admin(username, password)
How to Run
1. Import project into Eclipse
2. Add MySQL Connector to WEB-INF/lib
3. Configure Tomcat Server
4. Create MySQL database and tables
5. Run the project and open http://localhost:8080/StudentPortalWeb
